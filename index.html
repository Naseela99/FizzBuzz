What is FizzBuzz?
Fizzbuzz is a starter problem solving exercise. For a given number, we have to output all numbers starting from 1. If the number is divisible by 3, we output "FIZZ", if it is divisible by 5 we output "BUZZ". If the number is divisible by both 3 and 5, we output "FIZZBUZZ". Otherwise we output the number.
